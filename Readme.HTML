Virtual Store

##Aplicativo React Native: Virtual Store Este é um aplicativo simples que permite aos usuários escolher em uma lista de opções suas figurinhas favoritas para adicionar no carrinho de compras. Ele foi criado usando React Native e a API do https://rickandmortyapi.com/. ##Um pequeno disclaimer: Inicialmente o app havia side desenvolvido em cima da api REST COUNTRIES https://restcountries.com/ porem, a mesma se via fora do ar durante todo o domingo. Para que não houvesse atrasos e nem incertezas na hora da execução e avalição do app, foi decidido a troca para uma api mais estável e que não mudasse as caracteriscas do app sendo ainda uma loja virtual de aquisição de produtos. De qualquer forma ainda é possivel encontrar a primeira versão com a api do REST COUNTRIES na branch chamada "chore/refactory-styles" : ##Pré-requisitos Antes de executar o aplicativo, certifique-se de ter instalado o Node.js e o React Native CLI. ##Como baixar e executar o aplicativo Clone o repositório do aplicativo:

Requisitos Antes de prosseguir, certifique-se de ter os seguintes requisitos instalados em sua máquina: - Node.js (versão 12 ou superior) - Git - Yarn (opcional, mas recomendado)

  • 1. Clone o repositório do aplicativo: - HTTPS: https://github.com/otherRick/virtual-store.git - repositório: https://github.com/otherRick/virtual-store
  • 2. Navegue até o diretório do aplicativo: ...cd VirtualStore
  • 3. Instale as dependências do aplicativo: yarn install
  • 4. Instale as dependências do aplicativo IOS: - Entre na pasta ios: cd ios - Instale as dependencias: pod install - Retorne ao arquivo principal: cd ../
  • 5. Execute o comando para inciar o app: yarn start
    • ##Explique como usar o aplicativo: Na view "Home" do app, é possivel optar por qualquer card quantas vezes quiser em uma list infinta que carregará novos card conforme é escrollada para baixo. Cada item selecionado somara dentro do carrinho na parte superior do app em seu "Header". Clicando no icone do carrinho no header da view, ele redirecionará para a view de "Cart" onde pode-se visualizar todos os item selecionado na "Home", bem como: o valor multiplicado por unidade dos item, o valor total, as opções de redução ou exclusão individual e, por fim a opção de limpar o carrinho excluindo todos os itens. Caso, não haja nenhum item no carrinho, uma mensagem te orientará a voltar a view de home, dando assim, reinicio a todo processo. ## Features desejadas e não implementadas: Apesar de ter todas as funcionalidades requeridas em funcionamento, assim como o seu teste e story book, ainda assim algumas features que estavam no planejamento do app não foram implentadas.
